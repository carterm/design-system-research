# `cal-ds-cardgroup`

A simple cardgroup that wraps cards and maintains their height.

Sample usage:

```html
<cal-ds-cardgroup data-orientation="media-" data-action="interactive">
  <cal-ds-card>
    <template>
      <h2>
        <a href="#">Card Test 1</a>
      </h2>
      <p>This is only a test.</p>

      <img
        src="https://live.staticflickr.com/5609/15596534729_9c240ff83a_b.jpg"
        alt="Brown bear" />
    </template>
  </cal-ds-card>
  <cal-ds-card>
    <template>
      <h2>
        <a href="#">Card Test 2</a>
      </h2>
      <p>This is only another test.</p>

      <img
        src="https://live.staticflickr.com/5609/15596534729_9c240ff83a_b.jpg"
        alt="Brown bear" />
    </template>
  </cal-ds-card>
</cal-ds-cardgroup>
```
